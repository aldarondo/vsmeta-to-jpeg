# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-28

### Added
- Initial release.
- Support for extracting poster and backdrop images from .vsmeta files.
- Batch processing support for directories.
- CLI options: `--dry-run`, `--overwrite`, `--verbose`.
- Programmatic API `convertVsMetaToJpeg`.
